Project Euler solutions - DoggoPwn

Each solution contains PDF files with a mathemtical solution, a solution in JavaScript (high-level), C (low-level) and x64_86 Linux Assembly (lowest level).

For each problem I try to find a pattern, find a closed-form formula for iterative exercises, or at least simplify as much possible.


If an exercise is not listed here, try doggopwn.github.io/euler/X where X is the number of the exercise.
